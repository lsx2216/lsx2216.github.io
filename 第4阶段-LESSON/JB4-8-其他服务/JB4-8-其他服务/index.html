JB4-8-其他服务

[!NOTE] Java道经第4卷 - 第8阶 - 其他服务 lesson-monitor

S01. 开发监控服务

武技: 创建子项目 lesson-monitor 做为监控服务子项目

E01. 开发监控服务端

1. 第三方依赖

pom.xml

<!--spring-boot-admin-server-->
<dependency>
    <groupId>de.codecentric</groupId>
    <artifactId>spring-boot-admin-server</artifactId>
    <version>2.2.4</version>
</dependency>
<!--spring-boot-admin-server-ui-->
<dependency>
    <groupId>de.codecentric</groupId>
    <artifactId>spring-boot-admin-server-ui</artifactId>
    <version>2.2.4</version>
</dependency>

2. 开发主配项

classpath:application.properties

############################# App ##############################  
#端口号，项目名，管控台账号，管控台密码#  
server.port=5002  
spring.application.name=lesson-monitor  
spring.security.user.name=monitor  
spring.security.user.password=monitor

3. 开发启动类

`com.lsx.LessonMonitorApp

@EnableAdminServer  
@SpringBootApplication  
public class LessonMonitorApp {  
    public static void main(String[] args) {  
        SpringApplication.run(LessonMonitorApp.class, args);  
    }  
}

E02. 开发监控客户端

武技: 使用子项目 lesson-admin 和 lesson-server 做为监控客户端

1. 第三方依赖

pom.xml

<!--spring-boot-admin-starter-client-->  
<dependency>  
    <groupId>de.codecentric</groupId>  
    <artifactId>spring-boot-admin-starter-client</artifactId>
    <version>2.2.4</version>
</dependency>

2. 开发主配项

classpath:application.properties

############################# SpringBootAdmin ##############################  
#客户端地址，服务端地址，使用IP地址，账号，密码，暴露所有端点，展示health端点详情#  
spring.boot.admin.client.instance.service-base-url=http://localhost:5266  
spring.boot.admin.client.url=http://localhost:5002  
spring.boot.admin.client.instance.prefer-ip=true  
spring.boot.admin.client.username=monitor  
spring.boot.admin.client.password=monitor  
management.endpoints.web.exposure.include=*  
management.endpoint.health.show-details=always

3. 测试客户端

  1. 启动用户后台服务和监控服务，访问监控管控台。
  2. 在应用墙中查看监控的项目，在监控界面的 Insights -> 细节 右侧查看。

E03. 部署监控服务

武技：在Docker中部署 lesson-monitor 项目

1. 打包项目

  1. 将主配中的全部IP地址由 localhost 改为 192.168.40.77 虚拟机地址。
  2. 在子项目 lesson-monitor 中添加打包插件：

pom.xml

<!--项目打包-->
<build>
    <!--最终Jar包名-->
    <finalName>lesson-monitor</finalName>
    <plugins>
        <!--SpringBoot打包插件-->
        <plugin>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-maven-plugin</artifactId>
            <version>2.4.8</version>
            <!--配置启动类-->
            <configuration>
                <mainClass>com.lsx.LessonMonitorApp</mainClass>
            </configuration>
            <!--打包本项目全部依赖包到最终Jar包内-->
            <executions>
                <execution>
                    <goals>
                        <goal>repackage</goal>
                    </goals>
                </execution>
            </executions>
        </plugin>
    </plugins>
</build>
  1. 打包父项目（然后打包子项目）：
# 打包父项目
cd D:\workspace\java\lesson-project
mvn clean install -Dmaven.test.skip=true

2. 创建容器

  1. 创建相关目录：
# 创建相关目录
mkdir -p /opt/app/lesson/lesson-monitor;
chmod -R 777 /opt/app/lesson;
  1. 将Jar包 lesson-monitor.jar 拷贝到 /opt/app/lesson/lesson-monitor 目录中。
  2. 创建DockerFile：
# 开发节点的Dockerfile文件
cd /opt/app/lesson/lesson-monitor;
touch Dockerfile;
echo 'FROM openjdk:11' >> Dockerfile;
echo 'MAINTAINER lsx' >> Dockerfile;
echo 'ADD lesson-monitor.jar lesson-monitor.jar' >> Dockerfile;
echo 'EXPOSE 5002' >> Dockerfile;
echo 'CMD ["nohup", "java", "-jar", "lesson-monitor.jar", "&"]' >> Dockerfile;
  1. 构建镜像：
# 根据Dockerfile文件制作镜像: 必须在Dockerfile文件同级目录下
docker build -t lesson-monitor:v1.0 .

# 查看镜像
docker images
  1. 创建容器：
# 创建并运行节点容器
docker run --name lesson-monitor --network my-net -p 5002:5002 \
	-e TZ=Asia/Shanghai \
	-d lesson-monitor:v1.0;

# 查看节点容器
docker ps --format "{{.ID}}\t{{.Names}}\t{{.Ports}}"

# 永久开放5055端口
firewall-cmd --add-port=5002/tcp --permanent
firewall-cmd --reload
  1. 在Windows机器上 监控服务
