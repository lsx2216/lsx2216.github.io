E02. Java环境变量配置

心法: 为何要配置系统Path变量

  • Windows系统的 Path 变量专门存放某些软件的可执行程序所在的目录，一般为 bin 目录。
  • CMD在执行某个 .exe 程序的时候会优先在当前目录中寻找:
    • 若当前目录中没有这个 .exe 程序，则尝试在系统的 Path 变量中的全部目录下依次寻找。
  • 配置 Path 意义就是让CMD快速找到Java的可执行程序。

武技: 将JDK配置到系统Path变量中

  1. 拷贝Java家目录中bin目录的地址:
    • 如 D:\jdk\jdk11\bin 目录。
  2. 点击 右键此电脑 -> 属性 -> 高级系统设置 -> 高级 -> 环境变量 -> 用户变量:
    • 若无Path则新建Path，添加 D:\jdk\jdk11\bin 值，点击确定完成新建。
    • 若有Path则编辑Path，追加 D:\jdk\jdk11\bin 值，千万不要将 Path 的原值删除。
  3. 重启CMD并在任意位置测试以下命令:
    • cmd: javac.exe: 负责编译Java文件的编译程序，.exe 后缀可省略。
    • cmd: java.exe: 负责运行Java文件的运行程序，.exe 后缀可省略。
    • cmd: set "JAVA_HOME": 查看JAVA_HOME变量。

E03. Java环境变量升级

心法: 为何要配置系统JAVA_HOME变量

  • 系统变量 JAVA_HOME 用于配置JDK的家目录，是自定义的。
  • 系统变量 JAVA_HOME 用于实现系统变量 Path 的归一原则:
    • 归一原则: 尽量将相同的，易变的内容抽取为变量，将变化归于一处，统一处理和调度。
  • 其它如Maven之类的程序可能会默认寻找该变量。

武技: 配置JAVA_HOME变量

  1. 拷贝Java家目录地址:
    • 如 D:\jdk\jdk11 目录。
  2. 点击 右键此电脑 -> 属性 -> 高级系统设置 -> 高级 -> 环境变量 -> 用户变量:
    • 新建JAVA_HOME变量，添加 D:\jdk\jdk11 值，点击确定完成新建。
  3. 优化Path变量，使其满足归一原则:
    • 修改Path变量的值为 %JAVA_HOME%\bin 即可。
    • 若Path变为单行模式，则在变量值最前面添加 C:\Windows\System64; 即可。

E04. Java入门程序开发

心法: Java程序运行三步骤

步骤动作描述
第1步编写我们写代码文件，叫做源文件，后缀是 .java
第2步编译把我们能看懂的 .java 文件编译成JVM能看懂的 .class 文件
第3步运行将 .class 文件中的全部类先加载到JVM，然后再转换成对应OS的机器码，最后运行机器码

武技: 开发HelloWorld程序

  1. 安装Nodepad3记事本：
    1. Nodepad3安装教程
  2. 编写源代码 D:\\HelloWorld.java 文件:
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}
  1. 编译源代码: 每次改动代码需要重新编译才可生效:
# 切换到D盘
d:

# 使用 `dir` 命令查看该位置下的文件
dir

# 编译HelloWorld.java文件
javac HelloWorld.java
  1. 运行字节码:
    • 运行是指运行加载到JVM中的类，而不是字节码文件本身，所以不要添加 .class 后缀。
    • JDK11的版本中，可以直接运行Java源文件，无需编译也不会产生字节码文件。
# 运行JVM中的 `HelloWorld` 类
java HelloWorld

# 运行JVM中 `HelloWorld` 包下的 `class` 类
java HelloWorld.class

E05. Java入门程序详解

心法: 编码三大规矩

  • Java代码中不会出现任何中文符号，字符串中却可以。
  • Java代码中的小括号，中括号，大括号，单引号，双引号都是成对儿出现的。
  • Java代码中，遇到左大括号，下一行必须缩格。

心法: 解析HelloWorld程序 - Java类

  • Java类是Java封装的最小单位。
组成部分中文描述
public修饰符表示该类可以被公开访问，一个Java文件中只能存在一个被 public 修饰的类
class关键字用来标识后面的 HelloWorld 是一个类一个Java文件中可以存在多个 class 关键字，一个Java文件可以编译成多个字节码文件
HelloWorld类名必须和文件名一致且不能重复，首字母大写驼峰制，建议使用英文名词
{}类体类体中只能包含属性和方法

心法: 解析HelloWorld程序 - main方法

  • main方法是程序主入口，只能写在 public class 中才可以被调用。
组成部分中文描述
public修饰符表示该方法可以被公开访问，以保证JVM在任何情况下都可以访问到这个入口
static修饰符表示该方法共享唯一先行，为了让JVM调用起来更方便
void返回值表示main方法不返回任何值，main方法结束即表示整个程序结束，无需返回值
main方法名JVM只认 main 为入口方法的名字，所以该名称是固定的
(String[] args)形参列表可使用 java HelloWorld aa bb 来向main方法传递参数
{}方法体方法体中只能包含逻辑代码

心法: 解析HelloWorld程序 - 输出语句

  • 输出语句用于在控制台输出指定的内容。
组成部分中文描述
System.out系统标准输出默认输出为值为 Console 控制台
.println("HelloWorld!")换行输出方法括号中的内容就是输出的内容
;代码结束标记Java中的每条语句后，只要不以 } 结尾，就必须要以 ; 结尾

E03. IDEA基础环境配置

心法: IDEA的常用配置入口

  • IDEA当前项目常用的配置项入口:
    • File -> Settings
  • IDEA后续项目常用的配置项入口:
    • File -> New Projects Setup -> Settings for New Projects..

武技: 禁用IDEA自动更新

  1. 进入 File -> Settings -> Appearance & Behavior -> System Settings -> Updates 界面。
  2. 取勾 Check IDE updates for .. 选项。
  3. 取勾 Check for plugin updates 选项。
  4. 取勾 Show What's New in the editor after an IDE update 选项。

武技: 设置调整IDEA字体大小的快捷键

  1. 进入 File -> Keymap 界面。
  2. 展开 Editor Actions 栏，在右上角搜索栏中输入 font 关键字。
  3. 选择 Increase Font Size，设置快捷键为 Ctrl+Wheel up，即 Ctrl 加鼠标上滑轮。
  4. 选择 Decrease Font Size，设置快捷键为 Ctrl+Wheel down，即 Ctrl 加鼠标下滑轮。
  5. 选择 Reset Font Size，设置快捷键为 Ctrl+Middle-Clidk，即 Ctrl 加鼠标中键。

武技: 设置IDEA字符编码（需要再次设置其他项目）

  1. 进入 File -> Settings -> Editor -> File Encodings 界面。
  2. 修改 Global Encoding 为 UTF-8 编码。
  3. 修改 Project Encoding 为 UTF-8 编码。
  4. 修改 Default encoding for properties files 为 UTF-8 编码。

武技: 设置IDEA代码模板

  1. 进入 File -> Settings -> Editor -> File and Code Templates 界面。
  2. 修改 Class/Interface/Enum/AnnotationType 的定义上方添加文档注释以及 @author 作者名。

武技: 隐藏IDEA文件目录

  1. 进入 File -> Settings -> Editor -> File Types 界面。
  2. 切到 Ignored Files and Folders 选项卡。
  3. 添加 *.idea 以隐藏Project视图中的 *.idea 目录，按回车确认。
  4. 添加 *.iml 以隐藏Project视图中的 *.iml 文件，按回车确认。

武技: 启用IDEA自动导包

  1. 进入 File -> Settings -> Editor -> General - Auto Import 界面。
  2. 勾选 Add unambiguous imports on the fly: 动态添加明确的导入。
  3. 勾选 Optimize imports on the fly: 优化导入，即删除多余的包。

武技: 展示方法分隔线

  1. 进入 File -> Settings -> Editor -> General - Appearance 界面。
  2. 勾选 Show method separators 选项。

武技: 修改IDEA终端

  1. 进入 File -> Settings -> Tools -> Terminal 界面。
  2. 选择 Shell path 为 C:\Windows\system32\cmd.exe 即可。

仓库分类描述

远程仓库一般用于多地团队开发，可以内网配置或外网配置

    • 新建 D:\repository 仓库目录，位置和取名随意。
