S01. H5概念入门

E01. H5概念入门

心法: 超文本标记语言 Hyper Text Mark-up Language 5，简称HTML5或H5

  • H5是一门面向浏览器的标记语言，包括文本，图片，音频，视频，链接等超文本元素，用于布局HTML页面。
  • H5文件以 .html 为后缀，由标签和文本组成:
    • H5标签描述功能: 由 <> 和小写英文字母组成，分单标签如 <br /> 和双标签如 <p>文本</p> 两种结构。
    • H5文本描述内容: 支持且仅支持 <!-- 注释内容 --> 这一种注释格式，且不能嵌套。

1. 页面结构

心法: H5页面的基本结构

标签中文描述
<!DOCTYPE html>文档声明告诉浏览器我是一个HTML文档，一般写在首行
<html lang="zh-cmn-Hans"></html>根级标签告诉浏览器HTML代码从哪里开始，到哪里结束，简体中文环境
<head>页头标签开发功能性代码，可存在N个
<title页面标题网页被加入收藏夹或书签时会默认使用该标题作为名字
<meta charset="UTF-8" />编码配置设置页面编码为UTF-8
<body>页体标签用于开发展示性代码，可存在N个

2. 起始项目

武技: 创建War包子项目 v2-7-web-html，该项目用于学习HTML技术

  1. 将ICO图标文件 my.ico 拷贝到项目的webapp目录下。
  2. 在webapp目录下页面 view/基础结构.html:
<!--文档声明用于告诉浏览器我是一个HTML文档，一般写在首行-->
<!DOCTYPE html>
<!--根标签用于告诉浏览器HTML代码从哪里开始，到哪里结束-->
<!--lang="zh-cmn-Hans": 告诉浏览器本页面是中文简体环境，供翻译使用-->
<html lang="zh-cmn-Hans">
<!--页头: 开发功能性代码，可存在N个-->
<head>
    <!--页面标题: 网页被加入收藏夹或者书签时会默认使用标题作为名字-->
    <title>我的首页</title>
    <!--配置网页的字符编码: 建议使用UTF-8编码-->
    <meta charset="UTF-8">
    <!--配置网站关键字/网站作者/网站内容描述: 一般用于搜索引擎-->
    <meta name="keywords" content="Java，学习">
    <meta name="author" content="lsx">
    <meta name="description" content="该仅用于页面用于HTML学习">
    <!--统一路径前缀: 该配置对当前页中的全部href/src/action属性都生效-->
    <base href="/html/">
    <!--引入ico图标: 图标会显示在网站标题左侧-->
    <link rel="icon" type="image/x-icon" href="/my.ico">
</head>
<!--页体: 开发展示性代码，可存在N个-->
<body>
Hello HTML !
</body>
</html>
  1. 部署项目，启动Tomcat服务并访问 项目。
  2. 查看源码：在浏览器的开发者模式中可以查看代码结构，浏览器右键可以查看网页源代码。

1. 全局属性

心法: H5全局属性

  • H5标签中通过KV来描述属性，属性名小写，属性值建议使用单/双引号:
全局属性描述
id=""为标签设置不可重复的唯一标识，一般用于JS捕获
name=""为标签设置可重复的名字，一般用于表单传值
class=""为标签设置可重复的标识，一般用于CSS捕获
title=""用于设置当鼠标悬停在其所在标签上时，给出的文字提示内容
contenteditable="false"鼠标点击时，是否可以直接修改内容
hidden="none"隐藏元素
dir="ltr"设置文字显示方向，如 rtl/ltr 等

E02. H5文字标签

心法: H5文字标签

标签中文描述内容特点
<strong>加粗加粗文本可以直接替换H4的 <b>
<em>倾斜倾斜文本可以直接替换H4的 <i>
<ins>下划线下划线文本可以直接替换H4的 <u>
<mark>高亮高亮显示的文本默认效果是一个黄色的背景高亮
<del>删除线废弃的文本效果在文字上添加一条横线
<br />换行换行符，不显示HTML代码中无论回车多少次的效果都仅为一个空格
<hr />水平线一条水平线默认占所在父元素的100%宽度

<small>缩小缩小一个字号的文本
<sup>上标上标符号，如平方数字等缩小一个字号，且垂直位置上移
<sub>下标下标符号，如log底数等缩小一个字号，且垂直位置下移

S04. H5表单标签

E01. H5表单区域

心法: H5表单区域

表单区域相关标签描述
<form action="" method="">表单区域，用于采集用户信息，配合按钮元素可以将数据提交到后台:属性 action: 提交地址，可选URI或本地，默认 # 当前页。 属性 method: 提交方式，可选 GET 或 POST，默认 GET。属性 target: 提交目标，可选 _self/_blank/帧窗口名称 等，默认 _self。属性 autocomplete: 自动完成提示，可选 off/on，默认 on。
<filedset> -> <legend>组名</legend>对表单中的控件进行分组，整体可省略。
<input type="text" name="" />输入控件的外层建议使用 <label> 关联控件描述文字。
<button type="submit">提交</button>提提按钮用于将采集到的用户信息发送到后台，本身不携带任何表单数据。提交按钮，支持 formaction/formmethod 等属性重写表单对应属性。可选 type="reset": 用于将表单内的所有 value 重置回默认值。可选 type="button": 用于配合JS产生响应事件。

E05. H5输入类型

心法: 输入控件 <input> 常用类型

  • <input> 控件的类型由 type 属性决定，默认为 text 类型。
input类型描述
type="text"单行文本控件
type="password"单行密码控件，隐藏用户敲入的内容
type="radio"单选控件，需用 name 属性划分组别，默认值 on，支持 checked 默认选中
type="checkbox"多选控件，需用 name 属性划分组别，默认值 on，支持 checked 默认选中
type="hidden"隐藏控件，用于开发人员偷偷向后台传递数据用
type="file"文件控件，上传文件名时支持用 accept 规定接收的MIME类型
type="number"数字控件，只允许输入数字，支持 min/max/step 范围属性
type="date"日期控件，格式为 yyyy-MM-dd
type="time"时间控件，格式为 hh:mm:ss
type="datetime-local"日期时间控件，格式为 yyyy-MM-dd hh:mm:ss
