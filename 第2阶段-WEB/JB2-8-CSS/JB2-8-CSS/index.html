S01. C3基础入门

E01. C3基础入门

心法: 层叠样式表3 Cascading Style Sheets 3，简称CSS3或C3

  • CSS用于控制网页的样式和布局，包括字体、颜色、间距、边框、背景等方面的表现。
  • CSS与HTML结合使用，为网页添加样式和美化效果，使得网页内容更具吸引力和可读性。

武技: 创建War包子项目 v2-8-web-css，该项目用于学习CSS技术

  1. 部署项目，启动Tomcat服务并 访问项目

1. 技术特点

心法: CSS3技术特点

技术特点描述
层叠性相同样式会覆盖，不同样式叠加。
继承性子标签会继承父标签的外观，如字体/颜色等，但不会继承父标签的布局，边框/边距等

2. 样式分类

心法: CSS3样式分类

C3分类优先级使用方式特点
内嵌最高直接在某标签的 style 属性中使用代码耦合度高，不推荐
内部高在页头的 <style></style> 中开发学习场景下推荐使用
外部低在页头用 <link rel="stylesheet" href=""/> 引入外部CSS文件开发场景下推荐使用
浏览器样式最低浏览器对标签设置的 一个基本样式可以直接覆盖

E02. C3选择器

心法: C3选择器

  • C3选择器指的是捕捉页面标签的方式，选择器越长效率越低。

1. 常用选择器

常用选择器选择内容示例
*选择所有标签，建议指定具体范围，否则会匹配到很多无关的标签body * {}
A选择所有A标签，要求A必须是一个正确的HTML标签名p {}
#A选择 id="A" 的标签，无容错功能，重复的id的会被全部选中#a {}
.A选择 class="A" 的标签，class 属性值可写多个，空格分隔.a {}
A[B]选择所有拥有 B 属性的A标签#a[name] {}
A[B=C]选择所有拥有 B 属性，且 B 的值为 C 的A标签#a[name=c] {}
A[B*=C]选择所有拥有 B 属性，且 B 的值中包含 C 的A标签#a[name*=c] {}
A, B选择所有的A标签和B标签，支持多个#a, .b {}
A B选择所有的A标签中的B所有标签，支持多级body div {}
A:nth-child(N)选择范围内的第N个A标签p:nth-child(2){}
A:checked选择所有被勾选上的A标签，具有动态效果input:checked{}
A:hover选择所有被鼠标经过的A标签，具有动态效果div:hover{}
A:focus选择所有获取了焦点的A标签，具有动态效果input:focus{}

2. 其它选择器

心法: 其他选择器

  • 下表中的N都是从1开始的:
其他选择器选择内容示例
A > B选择A中的所有B，要求B只能是子标签p > span{}
A + B若与A相邻的下一个同级标签是B，则被选中h1 + p{}
A ~ B选择A后面的同级所有B，不在乎是否相邻h1 ~ p{}
A::first-line选择A中首行内容，以浏览器为准，与HTML代码无关p::first-line{}
A::first-letter选择A中的第一个字p::first-letter{}
A::before{content: "B"}在A的文本前插入 B 值p::before{content:"A"}
A::after{content: "B"}在A的文本后插入 B 值p::after{content:"B"}
A:first-child如果A是第一个子标签，就被选中p:first-child{}
A:last-child如果A是最后一个子标签，就被选中p:last-child{}
A:only-child如果A没有兄弟标签，就被选中，不考虑类型问题p:only-child{}
A:only-of-type如果A没有同类型的兄弟标签，就被选中p:only-of-type{}
A:nth-last-child(N)如果A是倒数第N个子标签，就被选中p:nth-last-child(2)
A:nth-of-type(N)如果A是其所有同类型标签中的第N个子标签，就被选中p:nth-of-type(2)
A:nth-last-of-type(N)如果A是其所有同类型标签中的倒数第N个子标签，就被选中p:nth-last-of-type(2)
A:enable选择被启用的A标签input:enable{}
A:disabled选择被禁用的A标签input:disabled{}
A:default选择默认的A标签，无动态效果，仅首次被打勾时才会被选中input:default{}
A:valid选择通过验证的A标签，包括 requrired/pattern 等input:valid{}
A:invalid选择未通过验证的A标签，包括 requrired/pattern 等input:invalid{}
A:required选择有 required 属性的A标签input:required{}
A:optional选择无 required 属性的A标签input:optional{}
A:link未访问的A标签，一般为超级链接a:link{}
A:visited已访问的A标签，一般为超级链接a:visited{}
A:active鼠标激活A标签时触发，一般为超级链接a:active{}
A:not()否定选择div:not(.a){}
A:empty选择没有内容的空A标签div:empty{}
A:lang(en)选择 lang="en" 或者没有 lang 属性的A标签html:lang(en){}

心法: C3盒子变形样式表

    • transform: translate(100px) scale(1.5) rotate(45deg) skew(45deg)
可选值描述
none默认值，不变形

skew(-45deg)逆时针倾斜45度，以元素中心点以下的部分为"脚"，正值脚向右动，负值脚向左动
transform-origin:10px 10px重新设置变形参考点，默认元素中心点为原点

E04. 动画样式表

心法: C3盒子动画

  • transition 可以让一次 :hover 动画具有过渡效果，也可以同时设置:
    • transition: width 3s ease-in 1s, background-color 5s ease 0s
相关属性可选值描述
transition-propertywidth, height: 样式属性列表all: 全部样式属性，默认值none: 清除所有过渡效果在原始元素上指定过渡的样式属性
transition-duration1s, 0s: 过渡效果的持续时间列表在原始元素上指定过渡的持续时间时间列表需要和样式属性列表对应
transition-delay2s, 0s: 过渡效果的延迟时间列表在原始元素上指定过渡的延迟时间延迟时间需要和样式属性列表对应
transition-timing-functionease: 匀减速过渡到最终值，默认linear: 匀速过渡到最终值ease-in: 加速过渡到最终值ease-out: 减速过渡到最终值ease-in-out: 先加速，再减速过渡到最终值在原始元素上指定过渡的速度效果

E02. C3弹性布局

心法: C3弹性布局 Flexible Box，简称FlexBox

  • FlexBox是一种一维布局，只能处理一行或一列元素布局，分为主轴和副轴（交叉轴）。
  • FlexBox无法使用浮动相关的样式。

心法: C3弹性布局设置流程

  1. 在父容器中设置弹性效果 display: flex : 此时容器中的直系子元素就会变为弹性元素。
  2. 在父容器中设置布局方向，即设置主轴和副轴:
父容器种的布局方向描述
flex-direction: row设置从左到右的X轴为主轴，默认值
flex-direction: row-reverse设置从右到左的X轴为主轴
flex-direction: column设置从上到下的Y轴为主轴
flex-direction: column-reverse设置从小到上的Y轴为主轴
  1. 在父容器中使用 justify-content 设置子元素 主轴 对齐方式，常用可选值见下表:
主轴常用对齐方式描述
justify-content: flex-start子元素在当前主轴居左或居上，默认
justify-content: flex-end子元素在当前主轴居右或居下
justify-content: center子元素在当前主轴居中
justify-content: space-between子元素在当前主轴两端顶格对齐
justify-content: space-evenly子元素在当前主轴均匀分布，整体两侧间距为子元素之间间距
justify-content: space-around子元素在当前主轴均匀分布，整体两侧间距为子元素之间间距的一半
  1. 在父容器中使用 align-items 设置子元素 副轴 对齐方式，常用可选值见下表:
副轴常用对齐方式描述
align-items: flex-start子元素在当前副轴居左或居上，默认
align-items: flex-end子元素在当前主轴居右或居下
align-items: center子元素在当前主轴居中
  1. 设置子元素的伸缩属性: 即如何在主轴进行伸缩以适应弹性父容器中的可用空间:
    • flex: n: 该子元素占整个父容器的 n/子元素总个数 份，默认 0，表示不拉伸。
    • 设置了 flex 属性的子元素的宽高将自动进行伸缩。

E03. C3定位布局

心法: C3定位布局

  • 定位决定元素位置，先通过 position 决定方式，再通过 top / left 等决定位置:
  • 项目中推荐相对中绝对的组合方式，可以整合绝对定位和相对定位的优点:
    • 在父元素中添加 position: relative: 将父元素视为相对点，更灵活。
    • 在子元素中添加 position: absolute: 子元素绝对定位，不占地方。
定位方式中文参考点描述优点缺点
static无定位无默认值无无
absolute绝对定位文档原点会随文档滚动不干扰其他元素，相当浮在页面上层，支持使用 z-index: -1 调整到下层不灵活，不随父元素移动
relative相对定位自身原点会随文档滚动在移动前的位置留下一个透明分身，该分身会干扰其他元素灵活，会父元素移动
fixed固定定位视口原点不随文档滚动不干扰其他元素，相当浮在页面上层，支持使用 z-index: -1 调整到下层不灵活，不随父元素移动
